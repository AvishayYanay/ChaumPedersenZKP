# ChaumPedersenZKP
The Chaum Pedersen proof system (i.e., proof of same exponent), see Section 3.2 in Nigel Smart's book: https://www.cs.umd.edu/~waa/414-F11/IntroToCrypto.pdf

To run the code in this repo:

`cd ChaumPedersenZKP`

`python3 server.py`

Then, in another terminal
`python3 client.py`
